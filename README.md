\# Student Manager

Simple Student Manager demo used for Version Control Systems assignment.



\## Features

\- Add / view simple student records (demo)

\- Example CI via GitHub Actions

\- Docs (docs/), CONTRIBUTING, CODE\_OF\_CONDUCT
